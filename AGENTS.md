# AGENTS.md

## 담당 영역

- 이 디렉터리(`beyond404-web`)는 web 영역입니다.
- 작업자는 web 영역만 담당합니다.
- server 영역(`../beyond404-server`)은 직접 수정하지 않습니다.

## 협업 규칙

- server 코드는 다른 팀원이 담당하므로, web 작업 중 server 파일을 변경하지 않습니다.
- server 변경 사항은 매일 `pull` 작업을 통해 받아옵니다.
- web 작업에서 server 동작 확인이 필요하더라도, server 코드를 수정하지 않고 담당자와 협의합니다.
