# SwapIt 크레딧 산정 규칙 (v2.0)

## 1. 최종 공식

```
최종 크레딧 = 스크랩 가치 + min(신제품가 × 크레딧 비율, 신제품가 × 15%)
```

- **스크랩 가치**: 반납 가전의 금속 원자재 기반 Floor Price. 외관 상태·연식과 무관하게 항상 전액 보장.
- **신제품 연계 크레딧**: 구매할 신제품 가격 × 크레딧 비율. 단, 신제품가의 15%를 상한으로 제한.
- **Cap(15%)은 신제품 연계 부분에만 적용** — 스크랩 가치에는 적용 안 됨.

## 2. 스크랩 가치 계산

### 공식
```
스크랩 가치 = 제품 무게(kg) × Σ(금속별 비율 × 시세)
```

### 금속 시세 (2025년 기준, 수동 업데이트)

| 금속 | 시세 |
|------|------|
| 철 (고철) | 350원/kg |
| 알루미늄 | 1,800원/kg |
| 구리 | 9,000원/kg |

### 가전별 금속 비율 상수 (WEEE 논문 최고값 기준)

| 가전 | 철 | 알루미늄 | 구리 |
|------|-----|---------|------|
| 냉장고 | 45% | 7% | 5% |
| 세탁기 | 65% | 4% | 3% |
| 에어컨 | 42% | 25% | 18% |
| 전자레인지 | 55% | 6% | 4% |
| TV | 22% | 9% | 6% |

### 가전별 크기 등급별 평균 무게

| 가전 | 소형 | 중형 | 대형 |
|------|------|------|------|
| 냉장고 | 35kg | 65kg | 100kg |
| 세탁기 | 50kg | 70kg | 90kg |
| 에어컨 | 15kg | 30kg | 50kg |
| 전자레인지 | 10kg | 14kg | 20kg |
| TV | 8kg | 15kg | 30kg |

> 현재는 크기 등급 기반 평균값 사용. 추후 모델별 실제 무게 DB 연동 시 교체 예정.

## 3. 크기 등급 판단 기준 (반납 가전)

| 가전 | 소형 | 중형 | 대형 |
|------|------|------|------|
| 냉장고 | 300L 이하 | 300~600L | 600L 초과 |
| 세탁기 | 9kg 이하 | 9~16kg | 16kg 초과 |
| 에어컨 | 1톤 이하 | 1~2톤 | 2톤 초과 |
| 전자레인지 | 20L 이하 | 20~30L | 30L 초과 |
| TV | 43인치 이하 | 43~65인치 | 65인치 초과 |

크기 등급은 LLM이 전면 사진 + 후면 스티커 OCR(모델명) → 스펙 조회를 통해 자동 판단.

## 4. 크레딧 비율 매트릭스

### 신제품 등급 기준 (구매 가격 기준 자동 분류)

| 등급 | 기준 |
|------|------|
| 프리미엄 | 150만원 이상 (디오스, 오브제컬렉션 등) |
| 일반 | 50만원 ~ 150만원 |
| 보급형 | 50만원 미만 |

### 비율 테이블

| 신제품 등급 | 1회 이용 (최초) | 2회 이용 (실버) | 3회+ 이용 (VIP) |
|------------|--------------|--------------|---------------|
| 프리미엄 | 10% | 12% | 15% |
| 일반 | 7% | 10% | 12% |
| 보급형 | 4% | 7% | 9% |

- 이용 횟수는 `users.swap_count` 컬럼 기준
- 상한선: 신제품가의 **15%** (어떤 경우에도 초과 불가)

## 5. 계산 예시

### 케이스 A — 최초 고객, 중형 냉장고 반납 → 프리미엄 냉장고 200만원 구매
```
스크랩:       65kg × 581원/kg = 37,800원
신제품 연계:  min(200만 × 10%, 200만 × 15%) = min(20만, 30만) = 200,000원
최종 크레딧:  37,800 + 200,000 = 237,800원
```

### 케이스 B — VIP(3회+), 중형 세탁기 반납 → 보급형 세탁기 30만원 구매
```
스크랩:       70kg × 444원/kg = 31,100원 (※ 스크랩은 cap 적용 안 됨)
신제품 연계:  min(30만 × 9%, 30만 × 15%) = min(2.7만, 4.5만) = 27,000원
최종 크레딧:  31,100 + 27,000 = 58,100원
```

### 케이스 C — 실버(2회), 중형 에어컨 반납 → 일반 에어컨 80만원 구매
```
스크랩:       30kg × 1,886원/kg = 56,600원
신제품 연계:  min(80만 × 10%, 80만 × 15%) = min(8만, 12만) = 80,000원
최종 크레딧:  56,600 + 80,000 = 136,600원
```

## 6. 사후 Lock-in 혜택 (AMC 골드 플랜)

SwapIt 교체 고객 전원에게 크레딧과 별개로 LG Best Care AMC 골드 플랜 무상 제공.

| 이용 횟수 | 제공 기간 |
|----------|---------|
| 1회 (최초) | 12개월 |
| 2회 (실버) | 18개월 |
| 3회+ (VIP) | 24개월 |

무상 종료 후 유료 구독(ARR) 모델로 자연 전환 유도.
